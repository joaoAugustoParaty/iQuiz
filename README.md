# 📱 iQuiz

Aplicativo iOS desenvolvido em Swift com o objetivo de criar uma experiência interativa de perguntas e respostas (quiz).

## 🚀 Sobre o projeto

O **iQuiz** é um aplicativo simples e escalável que permite ao usuário responder perguntas e testar seus conhecimentos de forma dinâmica.

Este projeto foi desenvolvido como parte dos meus estudos em desenvolvimento iOS utilizando o Xcode.

## 🛠️ Tecnologias utilizadas

* Swift
* UIKit 
* Xcode



## ▶️ Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/iQuiz.git
```

2. Abra o projeto no Xcode:

```bash
cd iQuiz
open iQuiz.xcodeproj
```

3. Clique em **Run (▶️)** ou pressione `Cmd + R`
